# 개인정보 처리방침

*최종 수정일: 2026년 3월 25일*

Unloop(이하 "회사")은 개인정보보호법 등 관련 법령에 따라 이용자의 개인정보를 보호하고, 이와 관련한 고충을 신속하게 처리하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

## 1. 수집하는 개인정보 항목

### 1.1 회원가입 시 수집

| 항목 | 수집 방법 |
|------|-----------|
| 소셜 로그인 식별자 (카카오/애플) | 소셜 로그인 시 자동 수집 |
| 닉네임 | 유저 직접 입력 |

### 1.2 서비스 이용 과정에서 수집

| 항목 | 설명 |
|------|------|
| 대화 내용 | AI 가이드와의 세션 대화 메시지 |
| 기분 기록 | 세션 시작 시 기분 슬라이더(0~10) 응답 |
| 심리 측정 결과 | PHQ-9, GAD-7 설문 응답 및 점수 |
| 세션 진행 상태 | 현재 Phase, 세션 번호, 커버된 개념 등 |
| 과제 이력 | 과제 내용, 완료 여부 |

### 1.3 자동 수집

| 항목 | 설명 |
|------|------|
| 기기 정보 | 기기 종류, OS 버전 |
| 접속 로그 | 접속 일시, 서비스 이용 기록 |

## 2. 개인정보의 이용 목적

- 서비스 제공: AI 기반 CBT 셀프케어 세션 진행
- 개인화: 유저의 진행 상태에 맞는 커리큘럼 제공
- 분석 리포트: 기분 추이, 측정 점수 변화 등 유저 개인 분석 제공
- 안전 프로토콜: 위기 상황 감지 및 전문 기관 안내
- 서비스 개선: 서비스 품질 향상을 위한 통계 분석 (비식별 처리)

## 3. 개인정보의 보유 및 파기

### 3.1 보유 기간

회원 탈퇴 시 수집된 개인정보는 **지체 없이 파기**합니다. 다만 관련 법령에 따라 아래 정보는 명시된 기간 동안 보관합니다.

| 보관 항목 | 보관 기간 | 근거 법률 |
|-----------|-----------|-----------|
| 계약 또는 청약철회 등에 관한 기록 | 5년 | 전자상거래 등에서의 소비자보호에 관한 법률 |
| 대금결제 및 재화 등의 공급에 관한 기록 | 5년 | 전자상거래 등에서의 소비자보호에 관한 법률 |
| 소비자의 불만 또는 분쟁 처리에 관한 기록 | 3년 | 전자상거래 등에서의 소비자보호에 관한 법률 |
| 접속 로그 (로그인 기록) | 3개월 | 통신비밀보호법 |

법적 보관 의무가 없는 항목(대화 내용, 기분 기록, 심리 측정 결과, 세션 진행 상태, 과제 이력)은 **회원 탈퇴 즉시 파기**합니다.

### 3.2 파기 방법

- 전자적 파일: 복구 불가능한 기술적 방법(덮어쓰기, 완전 삭제)으로 파기
- 종이 문서: 분쇄기를 이용하여 파쇄

## 4. 개인정보 처리 위탁

Unloop은 서비스 운영을 위해 아래와 같이 개인정보 처리 업무를 위탁하고 있습니다.

| 위탁받는 자 | 위탁 업무 내용 |
|-------------|---------------|
| Amazon Web Services, Inc. | AI 응답 생성을 위한 클라우드 데이터 처리 (AWS Bedrock) |

위탁받는 자는 위탁 업무 수행 목적 외에 개인정보를 처리하지 않으며, 위탁 계약 시 개인정보보호법 제26조에 따라 개인정보의 안전한 관리를 위한 사항을 계약서에 명시하고 있습니다.

## 5. 개인정보의 국외 이전

Unloop은 AI 응답 생성을 위해 이용자의 개인정보를 국외로 이전합니다.

| 항목 | 내용 |
|------|------|
| 이전받는 자 | Amazon Web Services, Inc. |
| 이전되는 국가 | 미국 (US East - Virginia) |
| 이전 일시 및 방법 | 서비스 이용 시 네트워크를 통해 실시간 전송 |
| 이전되는 항목 | 대화 내용 (AI 가이드와의 세션 메시지) |
| 이전받는 자의 이용 목적 | AI 모델을 통한 응답 생성 |
| 보유 및 이용 기간 | API 요청 처리 완료 즉시 삭제 (별도 보관하지 않음) |

<Note>
  AWS Bedrock은 전송된 입력 데이터를 모델 학습에 사용하지 않으며, 요청 처리 후 별도로 저장하지 않습니다.
</Note>

## 6. AI 학습에 대한 안내

- Unloop의 AI는 사전 학습된 언어 모델을 기반으로 동작합니다
- **유저의 대화 내용은 AI 모델의 추가 학습(fine-tuning)에 사용되지 않습니다**

## 7. 이용자의 권리 및 행사 방법

이용자(또는 법정대리인)는 언제든 다음 권리를 행사할 수 있습니다.

- 개인정보 열람 요청
- 개인정보 정정 요청
- 개인정보 삭제 요청 (회원 탈퇴)
- 개인정보 처리 정지 요청

권리 행사는 서비스 내 설정 화면 또는 아래 개인정보 보호 책임자에게 이메일로 요청할 수 있으며, 회사는 지체 없이 조치합니다.

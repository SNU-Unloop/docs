# 데이터 삭제 요청

Unloop(개발사: Huve)는 이용자의 데이터 삭제 요청권을 존중합니다.  
아래 두 가지 방법 중 하나로 삭제를 요청하실 수 있습니다.

---

## 방법 1 — 앱에서 직접 삭제 (즉시 처리)

1. **Unloop 앱** 실행
2. 하단 탭 **"홈"** → 우측 상단 **프로필 아이콘** 탭
3. **설정** → **계정** → **"계정 삭제"** 선택
4. 삭제 확인 → 모든 개인 데이터 **즉시 영구 삭제**

---

## 방법 2 — 이메일로 요청

앱 접근이 어려운 경우 아래 이메일로 요청해 주세요.  
요청 접수 후 **7영업일 이내** 처리 완료 및 확인 메일을 발송합니다.

- **이메일**: unloopbyhuve@gmail.com  
- **제목**: `[데이터 삭제 요청] 가입 이메일 주소`  
- **본문**: 가입 시 사용한 이메일 주소 기재

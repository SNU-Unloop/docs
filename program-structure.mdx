## 프로그램 개요

Unloop은 **하루 1세션, Phase 기반 구조화 프로그램**입니다. 각 Phase는 명확한 학습 목표를 가지며, 유저의 진행 상태에 따라 순차적으로 진행됩니다.

## Phase 구조

| Phase | 이름 | 핵심 목표 | 주요 방법론 |
|-------|------|-----------|------------|
| Phase A | 감정 알아차리기 | 감정 메커니즘 이해, RNT 루프 관찰 기초 | 심리교육, 3요소 모델, 마음챙김 기초 |
| Phase B | 인지적 유연성 | 자동적 사고 검토, 인지 재구성 | CBT 사고 기록, 소크라테스식 질문 |
| Phase C | 행동 변화 | 회피 식별, 행동 실험, 반대 방향 행동 | 행동 활성화(BA), 행동 실험 |

## 세션 구성

각 세션은 Beck의 CBT 6단계 세션 구조를 기반으로 합니다.

1. **기분 체크인** — 세션 시작 시 현재 기분 측정 (0~10 슬라이더)
2. **안건 설정** — 오늘 세션에서 다룰 주제 확인
3. **핵심 작업** — 개념 학습 또는 실습 (Phase별 기법 적용)
4. **요약** — 세션에서 다룬 내용 정리
5. **과제 합의** — 다음 세션까지의 과제를 함께 설정
6. **피드백** — 세션에 대한 유저 피드백

## 측정 도구

| 도구 | 용도 | 시점 |
|------|------|------|
| 기분 슬라이더 (0~10) | 매 세션 기분 추적 | 세션 시작 시 |
| PHQ-9 | 우울 증상 측정 | 온보딩 + 매 14세션 |
| GAD-7 | 불안 증상 측정 | 온보딩 + 매 14세션 |

## 과제(숙제) 원칙

과제는 일방적으로 부여되지 않으며, AI 가이드와 유저가 함께 설정합니다.

- **1단계**: 관찰/인식 — "~할 때 알아차려보세요"
- **2단계**: 1회 시도 — "한 번만 해보세요"
- **3단계**: 반복 — "이번 주 N번 해보세요"

첫 도입 세션에서 실행과 기록을 동시에 요구하지 않습니다.

## 프로그램 개요

Unloop은 하루 1세션, 총 100세션으로 구성된 Phase 기반 구조화 프로그램입니다. 각 Phase는 명확한 학습 목표를 가지며, 유저의 진행 상태에 따라 순차적으로 진행됩니다.

## Phase 구조

| Phase | 이름 | 세션 | 핵심 목표 | 주요 방법론 |
|-------|------|------|-----------|------------|
| Phase A | 감정 알아차리기 | S1~S13 | 감정 메커니즘 이해, RNT 루프 관찰 기초 | 심리교육, 3요소 모델, 마음챙김 기초 |
| Phase B | 인지적 유연성 | S14~S24 | 자동적 사고 검토, 인지 재구성 | CBT 사고 기록, 소크라테스식 질문 |
| Phase C | 행동 변화 | S25~S40 | 회피 식별, 행동 실험, 반대 방향 행동 | 행동 활성화(BA), 행동 실험 |
| Phase D | 수용과 심리적 유연성 | S41~S70 | 탈중심화, 수용, 인지적 탈융합 | MBCT, ACT, 마음챙김 명상 |
| Phase E | 통합과 자율적 유지 | S71~S100 | 도구 통합, 자기치료자화, 재발 방지, 졸업 | 전 패러다임 통합 |

### Phase별 핵심 전환

- <strong>Phase A→B</strong>: 감정을 알아차리는 법 → 사고의 내용을 검토하는 법
- <strong>Phase B→C</strong>: 사고를 바꾸는 법 → 행동으로 검증하는 법
- <strong>Phase C→D</strong>: "생각을 바꾸는 법을 배웠으니, 이제는 바꿀 수 없는 생각도 안고 갈 수 있는 법을 배웁니다"
- <strong>Phase D→E</strong>: "어떤 상황에서 어떤 도구를 쓸지 스스로 선택하는 연습. 목표는 앱 없이도 할 수 있는 상태입니다"

### AI 가이드 역할의 변화

| Phase | AI 역할 |
|-------|---------|
| Phase A~B | 교육자 — 개념을 안내하고 안전감 조성 |
| Phase C~D | 협력자 — 부드러운 질문으로 자기 발견 유도 |
| Phase E | 컨설턴트 → 옵서버 — 유저의 독립적 선택을 지지 |

## 세션 구성

각 세션은 Beck의 CBT 6단계 세션 구조를 기반으로 합니다.

1. 기분 체크인 — 세션 시작 시 현재 기분 측정
2. 안건 설정 — 오늘 세션에서 다룰 주제 확인
3. 핵심 작업 — 개념 학습 또는 실습
4. 요약 — 세션에서 다룬 내용 정리
5. 과제 합의 — 다음 세션까지의 과제를 함께 설정
6. 피드백 — 세션에 대한 유저 피드백

## 측정 도구

| 도구 | 용도 | 시점 |
|------|------|------|
| 기분 슬라이더 (0~10) | 매 세션 기분 추적 | 매 세션 시작 시 |
| 주 증상 체크인 | 우울/불안/둘 다 분기 선택 | 온보딩 (S1) |
| PHQ-9 | 우울 증상 측정 (9문항) | 온보딩 + 약 14세션 간격으로 반복 |
| GAD-7 | 불안 증상 측정 (7문항) | 온보딩 + 약 14세션 간격으로 반복 |

측정 결과는 분석 탭에서 점수 추이와 변화 해석을 확인할 수 있습니다.

## 과제 원칙

과제는 일방적으로 부여되지 않으며, AI 가이드와 유저가 함께 설정합니다.

- 1단계: 관찰/인식 — "~할 때 알아차려보세요"
- 2단계: 1회 시도 — "한 번만 해보세요"
- 3단계: 반복 — "이번 주 N번 해보세요"

첫 도입 세션에서 실행과 기록을 동시에 요구하지 않습니다.
